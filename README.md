# Python Open Source Audit

## Overview

This repository contains the required artifacts for **The Open Source Audit Capstone Project**, focused entirely on **Python**. It includes automated Linux shell auditing scripts and a structured technical setup that demonstrates Python's open-source philosophy, system integration, and Linux-based automation tools.

The project showcases **Linux scripting, Python environment inspection, log analysis, and open-source manifesto generation**.

## 🐧 Linux Execution Instructions

These scripts are designed **strictly for Linux systems**.

Running them on Windows or macOS without Linux environment may cause path and permission errors.

---

## 🔧 Prerequisites

* Linux OS or WSL
* Bash Shell
* Python3 installed
* Git installed
* Terminal access

---

## 🚀 Execution Steps

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/OpenSource_Linux_Python.git
cd OpenSource_Linux_Python
```

---

### 2️⃣ Give Execution Permission

```bash
chmod +x *.sh
```

---

### 3️⃣ Run Scripts

### System Identity Report

```bash
./system_identity.sh
```

Displays:

* Kernel
* User
* Uptime
* Distro
* Date
* Python License

---

### Python Package Inspector

```bash
./python_inspector.sh
```

Checks:

* Python installation
* Version
* License
* Package philosophy

---

### Python Environment Auditor

```bash
./python_env_auditor.sh
```

Audits:

* System directories
* Disk usage
* Permissions
* Python config directory

---

### Log File Analyzer

```bash
./pip_log_analyzer.sh
```

Analyzes:

* Log file
* ERROR count
* Last 5 matching lines

---

### Philosophy Generator

```bash
./philosophy_gen.sh
```

Generates:

* Open Source Manifesto
* User-based philosophy
* Output file

---
